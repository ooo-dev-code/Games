# Zelda Game

<h2>A small Zelda Clone game in a 3x3 map where you fight monsters to open a secret door to get the treasure.</h2>

# Features:
- <h2>Player</h2>
    - Animated Movement</br>
    - Attacks</br>
    - 2 skins</br>
    - Health</br>
    - Money</br>

- <h2>Game</h2>
    - Game engine</br>
    - Tile map</br>
    - Collisions</br>
    - Monsters</br>
    - Decorations</br>
